# Main Info:
 - A Small app made to easily download things
 - Requests to add or make things are allowed in my server

# Language Info:
 - C++ apps dont need built, they work right away
 - Python apps need dependencies installed (explains how and its easy to do)
 - JS apps need at least Node JS Installed (apps explain how and give links)

# Different Apps It Has:
# MEDIA TYPE APPS
 - Upscalers that actually increase quality and not just resolution (AI & Non AI) (fast)
 - Frame Interpolators (can increase fps of Videos and Gifs)
 - Clip Makers (make shorts from videos with custom settings)
 - Simple Free Fully Modular Video Editor (you can make your own effects in python)
 - File Size Shrinkers (reduce file size of Videos Gifs and Images without destroying Quality)
 - Extreme Compressors (as small as a Video Gif or Image can go in terms of file size)
 - ASCII Converters (5 different script to make videos into ASCII Animations)
 - Frame Alignment Fixers (fixes frame alignment issues caused by weaker systems, and badly made editors)
 - ReEncoders (reencode videos and gifs with various different encoders)
 - Storage Compressor (compress file size and save in .vault file for reduced storage)

# DOWNLOAD TYPE APPS
 - YouTube Downloader (download YouTube videos as MP3 or MP4 without any quality loss)
 - Reddit Downloader (download Videos Gifs and Images from any Subreddit or Reddit User)

# MISC APPS
 - Oneko For Windows (Linux desktop cat pet for windows, customizable sprites)
 - Screen Blocker (make images appear and move on screen to make things harder)
 - Memory ID Logger (for modders)
 - Macro Recorder (not useful for games, especially with loading times)
 - Click Sounds (plays sounds when u click or press keyboard buttons)

# PUBLIC APPS (NOT MADE BY ME)
 - FlowFrames Interpolator (adds more fps to Videos and Gifs)
 - DNSpy (manual dll viewer)
 - WinDerStat (check system storage and health)
 - X64dbg (open source assembly viewer)
 - VoiceMeter (audio mixer)
 - Discord WEBM Meme Maker (makes the "bouncy" webm memes for discord)
 - Free Windows Activator (MAS)
 - Process Hacker (better task manager)
 - GlassWire (Internet Traffic Monitor)
 - MSI Afterburner (NVIDIA Overclocking, and system monitoring)
 - CrystalDiskInfo (check disk health)
 - Open Hardware Monitor (system health tracker)
 - LightShot (screenshot tool set to keybind, and lets you edit your screenshot in the tool, when taking the screenshot)
 - SignalRGB (all in one rgb tool)
